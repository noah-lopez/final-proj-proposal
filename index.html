Project Proposal



“Escape from Planet Eshelman”

Marcus Lee, 25812787; Michael Lu, 25464293; Noah Lopez, 25709092

Summary

You wake up in a prison tower on an abandoned planet.
In order to escape, you must progress up through randomized levels all the way to the top of the tower.
At the top of the tower is a spaceship which you pilot to shoot baddies and escape the planet.

Problem Description

We wish to design a game that is optimized for web browsers (and mobile?), with reasonable AI for the non-player characters as well as randomization of levels such that each playthrough is unique. In this regard, WebGL provides a good framework to build games (in either 2D or 3D) that load quickly on browser and can be made for mobile as well. WebGL is built on top of JavaScript and GLSL, which from Project 3-2 we know loads quickly and processes shaders quickly as well. The main challenges then are designing the AI and the levels, 1. So that they work, and 2. That they are randomized to a reasonable degree.

Goals and Deliverables

1) We will deliver a fully functional game. However, we expect the quality of 3D models, and length of the game to possibly be somewhat lacking. At the very least, assets and levels load, and the player should be able to interact with the world (e.g. fight bad guys, pick up items, etc).
2) We hope to deliver a fully functional game with a wide variety of game modes, many levels and beautiful 3D models. In particular, we hope to implement both the dungeon/prison portion and the spaceship shooter portion, with working controls and proper interaction.
